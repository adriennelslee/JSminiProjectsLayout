Dear viewer,

This page encompasses the use of CSS grid to present some mini javascript projects. The aim was to try complete one a day, in hopes of gaining a better understanding of how Javascript in incorperated in a practical sense.

"If you build it, they will come"

Project 01 - A drum kit:
Press the keyboard letters & the relevant drum noise will play accordingly.

Project 02 - CSS & JS clock:
A simple clock that will give you the analogue time. Clicking on the button will display the digital time, at the time of the click.

Keep scrolling...

Project 03 - CSS variables with JS:
Using JS to allow for the CSS properties of a photo to be adjusted.
Properties include: spacing, blur, border and opacity.

Some calming sea & wave background noise YouTube

Project 04 - CSS grid image gallery:
Click on a photo to view it in larger format. Click on the close button (upper left), and view another one. Each time the page is refreshed the size & layout of the images will change.

Project 05 - Flex panel Gallery:
Gallery made with flexbox, click on the individual items to see what happens... Click again to return back to orginal state.

Project 06 - AJAX type ahead:
Type in the city or state to find out the population number there. This only works for cities and states of America.

To view more, visit here: Github
